The Curator is a web application that helps users discover new content based on their interests. It leverages machine learning algorithms and external APIs to provide personalized recommendations to users.

The system uses the cosine similarity algorithm to measure the similarity between user preferences and available content. Users can input their preferences and interests through a user-friendly interface, and The Curator generates recommendations based on these inputs.

The backend of the application seamlessly integrates with the front-end through Streamlit, a framework for building interactive web applications in Python. This ensures that The Curator is responsive and functional.

Using Python as the primary programming language for implementing The Curator offers several advantages due to its versatility, robust libraries, and strong community support.

Users can explore a wide array of entertainment and informational content, such as Movies, TV shows, Trending Music, Stocks, Books, and more.
